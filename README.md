# CentOS K6 SDK

Base image of CentOS for K6 SDK.

This images includes additional packages to the centos base image :
`tcsh`, `zsh`, `make`, `gcc-c++`,
`boost-devel`, `expat-devel`, `xerces-c-devel`, `git`, `camke3`.

* `centos7` : 7
